# team1-lib-back-app
Team 1 Library Backend Application Feb/Marc 2026


# Setup

## Install dependencies
```sh
npm install
```

## Create `.env`
Create a `.env` file in the project root and set your variables:

    ```env
    PORT=#####

    DB_NAME="library-management"
    DB_HOST=localhost
    DB_PORT=5432
    DB_USER=YOUR_PG_USERNAME
    DB_PASSWORD=YOUR_PG_PASSWORD
    ```

## Run the app
```sh
npm run dev
```
